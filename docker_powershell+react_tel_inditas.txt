Docker telepítése és indítása

Docker container indítása:

Ha már ott a csomag a docker programban, akkor ellenőrizd le, hogy fut-e? 
Ha még nincs ott: 
docker-compose build
docker-compose up -d

VS Code - powershell termináljában: 

Lépj be a docker könyvtár web/html mappájába és nyiss egy új powershell terminált:
docker ps
docker exec -it container_szám /bin/bash

Ezután létrehozhatod az új react projektet, vagy futtathatod a meglévőt!

npx create-react-app appneve

cd appneve

npm start
